This documentation describes the specification, testing, and operation of microgrid controllers for industrial microgrids. The content aligns with IEEE and IEC standards for controller functions, performance, and interoperability.

IEEE Std 2030.7-2017: IEEE Standard for the Specification of Microgrid Controllers

IEEE Std 2030.7-2017 defines the functional requirements for microgrid controllers. These functions are grouped into two categories:

Dispatch Functions

  • Balancing generation and load
  • Re-dispatching controllable resources
  • Executing external orders with setpoints and reconfiguration commands

Transition Functions

  • Planned islanding
  • Unplanned islanding
  • Resynchronization
  • Black start
The operator initiates dispatch commands and breaker operations from the control room. Voltage regulation is primarily achieved through BESS control actions, with optional support from capacitor banks or tap-changing transformers. Key performance indicators include dispatch objective compliance, device status, and voltage deviation.
Response accuracy and settling time are critical for dispatch functions. Accuracy and settling of BESS dispatch can be evaluated by adjusting controller parameters. Setpoint enforcement is verified at the BESS using measurements of real power, reactive power, voltage, and frequency. Performance evaluation includes steady-state error and transient behavior. Power quality monitoring covers voltage harmonic distortion, current harmonic distortion, and voltage sag.
Controllable loads such as HVAC systems and non-essential industrial equipment can be connected or disconnected based on grid conditions and power availability. Transition functions ensure stable operation during mode changes and unplanned islanding, maintaining voltage and frequency control and power balance. Black start functionality energizes the microgrid after a blackout, following defined parameters for delay, voltage level, and load pickup.

IEEE Std 2030.8-2018: IEEE Standard for the Testing of Microgrid Controllers

IEEE Std 2030.8-2018 provides guidelines for testing microgrid controllers, including planned islanding, resynchronization, and black start. Controllers implement reference generation for voltage, frequency, and power, breaker control, and PLL functions.

Data Monitoring Requirements

  • Sequence of events for switching device status, commands, and control outputs
  • Voltage and current waveforms for all phases sampled at a minimum of 1/4 cycle with 1% voltage accuracy
  • Time series voltage and current data sampled at 100 ms intervals with 1% voltage accuracy

Microgrid Controller

The microgrid controllers monitor the microgrid operation and execute different microgrid functions within their network. The microgrid controller commands the BESS with active and reactive power (in voltage control mode with voltage and frequency) setpoints. The microgrid controllers in this model comprise these functions:
  1. Reference frequency generation
  2. Reference voltage generation
  3. Breaker control
  4. Active and reactive power reference generation
  5. Planned islanding from main grid
  6. Resynchronization with the main grid
  7. Black start of the microgrid with load pick up

Control Modes and Transition Logic

Microgrid controllers support two operating modes:
  • Grid-Forming Mode: Maintains voltage and frequency during islanded operation
  • Grid-Following Mode: Synchronizes with the main grid when connected
Transition logic ensures smooth switching between modes using sequenced checks on angle, frequency, and voltage.

Black Start Sequence and Parameters

Black start involves energizing the microgrid after a blackout using the following sequence:
  1. Start BESS and establish nominal voltage and frequency
  2. Close internal bus breakers
  3. Pick up critical loads in stages
  4. Synchronize renewables
  5. Pick up remaining loads

Microgrid Controller Block

The Microgrid Controller subsystem comprises :
  1. PLL
  2. Microgrid supervisory control
  3. Breaker Control
  4. Reference Generation
  5. Microgrid control functions
The Microgrid supervisory control implements the transition and dispatch functions by using Stateflow™. The transition functions include planned islanding, unplanned islanding, resynchronization, and black start. The atomic charts of the transition functions implement the individual dispatch functions.
The Reference generation subsystem generates voltage, frequency, active power and reactive power references to the BESS. The Microgrid control functions comprise the planned islanding, resynchronization and black start models.
