Overview

This block models a synchronous generator operating in a grid-connected or microgrid environment. You can use this block with two selectable fidelity levels :
  • Detailed Model : Simulates generator dynamics using physical components, including a synchronous machine, IEEE standard exciter, and a prime mover with governor and inlet volume dynamics.
  • Simplified Model : Uses a constant torque source as the prime mover and a constant DC voltage source as the exciter for computational efficiency.

Block Description

The detailed model consists of three main components:
  1. Machine
  2. Automatic voltage regulator (AVR) and Exciter
  3. Prime mover and Governer

Machine

You can configure the synchronous machine block to match your desired rating by specifying parameters such as reactances, inertia, and damping. This block simulates the electrical and mechanical dynamics of the generator, including rotor angle, speed, electromagnetic torque, and stator currents.
AVR & Exciter
The AVR and exciter implement an IEEE standard excitation system. They regulate generator terminal voltage by adjusting field voltage. You can select the IEEE type and configure gains, limits, and response characteristics.
Prime Mover
The prime mover represents the turbine or engine driving the generator. It is modeled using time constant dynamics for:
  • Governor : Controls torque based on speed and power reference.
  • Main Inlet Volume : Captures fluid dynamics for realistic response.

Simplified Model

You can use the simplified model to abstract the generator behavior using :
  • Constant Torque Source : Represents prime mover delivering fixed mechanical power.
  • Constant DC Voltage Source : Represents exciter providing fixed field voltage.
