Overview

This block models a Battery Energy Storage System (BESS) connected to the grid or a microgrid. The block supports two types of grid interface controllers :
Grid-following (GFL):
  • The BESS synchronizes with the grid voltage and frequency using a Phase-Locked Loop (PLL).
  • Operates as a controlled current or power source.
  • Follows grid conditions and injects/absorbs active and reactive power based on setpoints.
  • Commonly used in grid-connected applications.
Grid-forming (GFM) :
  • The BESS establishes voltage and frequency reference for the local grid or microgrid.
  • Can operate in islanded mode or provide black-start capability.
  • Controls output voltage and frequency directly, supporting grid stability and resilience.
  • Suitable for microgrids, weak grids, or grid restoration scenarios.
You can choose either grid-following mode or grid-forming mode accroding to your requirements. This enables flexible simulation of various grid support and stability scenarios.
You can also use this block with two selectable fidelity levels :
  1. Detailed Model: Accurately simulates the BESS using physical components, including battery dynamics, controller, and converter (with selectable average or switching behavior).
  2. Simplified Model: Uses a controlled voltage source in the grid-forming mode and an abstract DC-AC converter in the grid-following mode

Detailed Model

Block Description

This block contains two subsystems
  1. Plant
  2. Controller

Plant

The plant subsystem contains a battery block connected to a converter. You can also select the converter’s fidelity—average or switching mode. Use the average model for faster simulations in system-level studies or when switching transients are not critical. Use the switching model detailed electromagnetic and harmonic studies.

Controller

You can choose between two control strategies :
  • GFL
  • GFM

Simplified Model

You can use the simplified model of a BESS for system level studies. This block supports both grid-forming and grid-following operation modes, each realized using simplified control structures to facilitate faster simulations and easier integration into larger grid models.
For more information on the simplified model, please refer to GFL BESS Simplified, GFM BESS Simplified
