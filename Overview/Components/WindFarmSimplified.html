Overview

This block models a wind farm using wind turbine power equation which operates in grid-following mode. You can use the simplified model to abstract the wind turbine behavior using standard power curve data and algebraic equations. The simplified model works both in MPPT mode and power reference mode. This block requires only typical wind turbine values as inputs, such as :
  • Rated wind speed, cut-in and cut-out wind speeds
  • Rated power
  • Rotor diameter or area
  • Power coefficient ()
  • Air density (default 1.225 kg/m³)

Functionality

This block assumes ideal or near-ideal tracking and conversion, neglecting dynamic effects and high-frequency behaviors.
This block operates in two modes namely Maximum Power Point Tracking (MPPT) and Power reference modes. If you choose the Operating Mode option on the block to Power Reference external inputs for active and reactive power references would appear. You can choose either of these modes according to your requirements. This block also limits the output current to the value defined by Maximum current of the converter parameter on the block mask.
The wind turbine model supports two methods for estimating the power coefficient ():
