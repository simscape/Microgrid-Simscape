Overview

This block implements a resynchronization control strategy for Grid-Forming Battery Energy Storage Systems (BESS) operating in a microgrid. Its primary function is to smoothly transition the microgrid from islanded operation back to grid-connected mode by aligning voltage and frequency references with the main grid before closing the point of common coupling (PCC).
Key Objectives:
  • Match microgrid voltage magnitude, phase, and frequency to the main grid.
  • Minimize transients during reconnection.
  • Ensure safe and stable synchronization without breaker stress or system instability.

Detailed Description

Block Functionality

The controller :
  • Monitors grid voltage and frequency at PCC.
  • Adjusts internal voltage and frequency references of the grid-forming BESS to match the grid.
  • Provides synchronization status and breaker close signal when alignment criteria are met.

Synchronization Logic
Monitors alignment conditions (Δf, ΔV, Δθ) and issues sync-ready signal when thresholds are satisfied.

Alignment Criteria for Breaker Closing

Before the PCC breaker can close, the following conditions must be satisfied :
1. Voltage Magnitude Difference (ΔV)
  • Definition : The difference between the microgrid voltage magnitude and the main grid voltage magnitude.
  • Requirement :
  • Purpose : Prevent large inrush currents when breaker closes.
2. Frequency Difference (Δf)
  • Definition : The difference between the microgrid frequency and the main grid frequency.
  • Requirement :
  • Purpose : Avoid sudden torque oscillations and system instability.
3. Phase Angle Difference (Δθ)
  • Definition : The difference in electrical angle between the microgrid voltage and the main grid voltage at PCC.
  • Requirement :
  • Purpose: Ensure breaker closes when voltages are nearly in phase to minimize transient power flow.
Synchronization Logic
Continuously monitors ΔV, Δf, and Δθ and when all three are within their respective thresholds for a defined settling time (e.g., 100–500 ms), the controller :
  • Sets Status = TRUE.
  • Issues Breaker Close Command to reconnect the microgrid safely.
