Overview

This block models a Type-4 wind farm which operates in a grid-following mode connected to a grid or a microgrid. You can use this block with two selectable fidelity levels :
  • Detailed Model: Accurately simulates the wind plant using physical components, including wind turbine aerodynamics, generator and converter dynamics, and control systems (speed, torque, current, and power control).
  • Simplified Model: Uses standard wind turbine power curves and algebraic equations to provide a computationally efficient approximation of plant behavior.

Block Description

This block contains two subsystems
  1. Plant
  2. Controller

Plant

The Plant subsystem includes a wind turbine block connected to a Permanent Magnet Synchronous Generator (PMSG). It also contains a back-to-back converter. You can also select the converter’s fidelity—average or switching mode. Use the average model for faster simulations in system-level studies or when switching transients are not critical. Use the switching model detailed electromagnetic and harmonic studies.

Controller

The Controller subsystem offers two controllers :
  • Machine side control : Controls the machine side converter
  • Grid side control : Controls the grid side converter
Pitch control is not modeled in the detailed model

Simplified Model

You can use the simplified model to abstract the wind turbine behavior using standard power curve data or algebraic equations. The simplified model works both in MPPT mode and power reference mode. This model requires only typical wind turbine values as inputs, such as :
  • Rated wind speed, cut-in and cut-out wind speeds
  • Rated power
  • Rotor diameter or area
  • Power coefficient ()
  • Air density (default 1.225 kg/m³)
For more information on the simplified model, please refer to Wind Farm Simplified
