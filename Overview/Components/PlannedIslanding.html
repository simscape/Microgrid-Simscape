Overview

This block implements planned islanding logic for a Grid-Forming Battery Energy Storage System (BESS) in a microgrid. When the system anticipates disconnection from the main grid (due to maintenance or operational strategy), this controller calculates active power () and reactive power () references for the BESS to compensate for the power previously supplied by the grid, ensuring a seamless transition to islanded operation.
Key Objectives:
  • Maintain microgrid stability during transition.
  • Avoid voltage and frequency disturbances when the Point of common coupling (PCC) breaker opens.
  • Ensure load-generation balance by adjusting BESS power references.

Detailed Description

Block Functionality

The controller :
  • Monitors real-time power flow at the PCC.
  • Calculates the grid contribution (, ) before islanding.
  • Generates and for the grid-forming BESS to replace the grid supply.
  • Coordinates with the BESS control mode (VF or PQ) for stable islanded operation.
