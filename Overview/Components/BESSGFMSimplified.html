Overview

This block models a Grid-Forming (GFM) battery energy storage system (BESS) model provides an abstracted representation of a GFM-BESS for system level studies. This block supports simplified control structures to facilitate faster simulations and easier integration into larger grid models.

Functionality

This block is modeled as an ideal controlled voltage source whose magnitude is controlled by the reactive droop control and phase is controlled by active droop control.
The key features of this block include :
  • Reduced-Order Modeling: Focuses on essential dynamic behavior, omitting detailed converter switching and inner control loops.
  • Droop control : The phase angle of the output voltage is adjusted based on the active power (P), enabling frequency support and reactive power sharing. The voltage magnitude is dynamically adjusted according to the reactive power (Q) using droop control logic, supporting grid voltage regulation and power sharing.
  • Simplified Converter Model : The battery is connected to the grid via a simplified converter that does not explicitly model switching or inner control loops.
  • Applications : Suitable for microgrid operation, black start, and voltage/frequency regulation studies.
This block operates either in voltage and frequency (V/F) mode or the active and reactive power (PQ) mode. In the V/F mode, voltage magnitude input to the controlled voltage is driven by the voltage droop, phase input is driven by the frequency droop. In the PQ mode, voltage magnitude input to the controlled voltage is driven by the reactive power droop, phase input is driven by the active power droop.
