Overview

This block models the mechanical power control path for synchronous machines driven by either :
  • Steam turbine (non-reheat), or
  • Diesel engine.
It combines :
  • A governor that regulates speed/frequency by adjusting energy input (steam valve or fuel rack).
  • A prime mover model that converts the energy input into mechanical power/torque using configurable time constants.
Applications:
  • Frequency control and primary response.
  • Microgrid islanding and resynchronization.
  • Generator dynamic studies for thermal and diesel units.

Detailed Description

Block Functionality

This block :
  • Compares rotor speed/frequency to reference and computes an energy input command.
  • Applies governor droop or isochronous control with PI logic.
  • Models prime mover dynamics using cascaded first-order lags:
  • Steam turbine: valve lag + turbine stage lag.
  • Diesel engine: fuel system lag and engine torque lag.
  • Outputs torque () to synchronous machine.
