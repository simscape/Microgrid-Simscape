Overview

This block implements a grid-following control strategy for Battery Energy Storage Systems (BESS). It generates reference signals for the BESS converter to operate in grid-following mode, ensuring synchronization with the grid and compliance with voltage and frequency requirements.
You can use this block for :
  • Active and reactive power control
  • Grid support functions (frequency and voltage regulation)
  • Microgrid and renewable integration studies

Detailed Description

Block Description

This block measures grid voltage and current, synchronizes with the grid using a Phase-Locked Loop (PLL), and computes reference signals for the BESS converter. It supports :
  • Active Power Control: Based on external power reference or droop characteristics.
  • Reactive Power Control: Based on voltage reference or reactive power setpoint.
  • Grid Synchronization: Maintains phase alignment with grid voltage.
