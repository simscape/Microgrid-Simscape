Overview

This block models a solar plant which operates in a grid-following mode connected to a grid or a microgrid. You can use this block with two selectable fidelity levels :
  1. Detailed Model: Accurately simulates the solar plant using physical components, including solar cell dynamics, MPPT (Maximum Power Point Tracking) controller, power reference controller, and converter (with selectable average or switching behavior).
  2. Simplified Model: Uses Anderson’s translational equations and typical datasheet values to provide a computationally efficient approximation of plant behavior.

Block Description

This block contains two subsystems
  1. Plant
  2. Controller

Plant

You can scale the solar cell block by specifying the number of panels in series and parallel, configuring the plant for your desired rating. You can also select the converter’s fidelity—average or switching mode. Use the average model for faster simulations in system-level studies or when switching transients are not critical. Use the switching model detailed electromagnetic and harmonic studies.

Controller

You can also choose between two control strategies :
  • MPPT Mode: Select 1 to enable Maximum Power Point Tracking.
  • Power Reference Mode: Select 0 to operate based on an external power reference.

Simplified Model

You can use the simplified model to abstract the solar plant behavior using Anderson’s translational equations. This model requires only typical datasheet values as inputs, such as :
  • Open-circuit voltage ()
  • Short-circuit current ()
  • Maximum power point voltage and current (, )
  • Temperature coefficients (α, β)

Anderson’s equations

Anderson’s translational equations are a set of empirical formulas that estimate the electrical output characteristics (current, voltage, power) of a photovoltaic (PV) module or array under varying environmental conditions. They provide a simplified yet effective way to predict PV performance based on standard test condition (STC) data sheet values, irradiance, and temperature, without requiring a full physical or equivalent circuit model.
You can use these equations when computational speed is critical, such as in large-scale grid studies or for preliminary PV system sizing.
