Overview

This block implements a power reference control strategy for a converter-based system (e.g., BESS or inverter) operating in PQ mode. It generates voltage references for the converter based on active power () and reactive power () commands and measured terminal voltage. The block uses inner current control, without an outer voltage loop.
Key Features :
  • Calculates dq current references from , , and measured voltage.
  • Regulates converter currents using PI controllers.
  • Outputs voltage commands for modulation.
Applications :
  • Grid-following converters in PQ mode.
  • BESS power dispatch.
  • Renewable integration with fixed PQ control.

Detailed Description

Block Functionality

This block :
  • Converts power references (, ) into dq current references using measured terminal voltage.
  • Implements inner current control loops to track dq current references.
  • Generates dq voltage commands for converter modulation.
