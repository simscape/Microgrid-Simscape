Overview

This block implements the Type-4 wind turbine control for a full-scale converter architecture. The generator typically permanent magnet synchronous generator (PMSG) is decoupled from the grid by two power converters :
  • Machine-side converter (MSC) : Runs field-oriented control (FOC) to regulate generator torque and rotor speed. It enables Maximum Power Point Tracking (MPPT) below rated wind and coordinates with pitch control above rated wind.
  • Grid-side converter (GSC) : Runs grid-following control aligned by a synchronous-reference-frame PLL to regulate DC-link voltage and provide reactive power / voltage support to the grid through dq current control

Detailed Description

Block Functionality

The controller :
  • Generates torque, current, and voltage references for MSC and GSC.
  • Coordinates Maximum Power Point Tracking (MPPT) for optimal power extraction.
  • Maintains DC-link voltage and reactive power support.

Control Structure

This table gives you the information about different units in the controller
Unit
Description
MPPT
Calculates rotor speed reference based on wind speed to maximize aerodynamic efficiency. Uses lookup table methods.
MSC Control
Regulates generator torque and speed. Outer loop sets torque reference from speed error; inner dq current loops control flux and torque using FOC.
GSC Control
Maintains DC-link voltage and reactive power. Outer loop regulates DC-link voltage; inner dq current loops generate voltage commands for grid compliance.
Fault Ride-Through
Implements LVRT logic with reactive current injection and torque reduction to protect DC-link and support grid voltage.

Machine-Side Converter (MSC) Control

  • Purpose : Controls generator torque and rotor speed for aerodynamic efficiency.
  • PMSM Current Controller : Generates voltage reference for MSC modulation

Grid-Side Converter (GSC) Control

  • Purpose: Maintains DC-link voltage and provides reactive power support while synchronizing with the grid.
  • Outer Loop : DC-link PI controller adjusts active current reference to maintain voltage.
  • Reactive Control : Sets q-axis current for reactive power or voltage support.
  • Inner Loops : dq current controllers generate voltage commands for modulation.
  • Output : Voltage reference for GSC modulation.
